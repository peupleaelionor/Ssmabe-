# public/assets/songi — Kit d'assets Songi Songi Mabé

Arborescence des assets de marque. Voir le plan complet :
[`docs/brand-assets.md`](../../../docs/brand-assets.md).

| Dossier | Contenu | Type dominant |
|---|---|---|
| `brand/` | charte exportée, nuancier, aperçu typo | référence |
| `logo/` | variantes de logo (SVG prioritaire) | **asset final** |
| `icons/` | favicon + app icon | **asset final** |
| `mascot/` | mascotte chat (head dark/light, full body) | **asset final** |
| `ui/` | repères UI | référence de style |
| `mockups/` | mockups mobile marketing | image finale |
| `social/` | kit réseaux (IG profile/post/story, X banner) | image finale |
| `marketing/` | hero card, quote card, feature card | image finale |
| `modes/` | référence cartes modes (recodées en composant) | référence |
| `security/` | référence trust banner + badges (recodés) | référence |

## Conventions
- **SVG** pour logo / mascotte / favicon / pictos (vectoriel, recolorable).
- **WebP** pour visuels web, **PNG** pour plateformes sociales.
- Nommage : `kebab-case`, suffixe `-dark` / `-light` pour les variantes de fond,
  suffixe `@2x` pour le retina.
- Les `.gitkeep` maintiennent les dossiers vides versionnés ; les remplacer par
  les vrais fichiers au fur et à mesure.
