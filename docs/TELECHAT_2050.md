# Songi Songi Mabé — Doctrine Téléchat 2050

## Principe absolu

**Devant : simplicité populaire. Derrière : architecture futuriste.**

L'utilisateur ne doit jamais avoir l'impression d'entrer dans une plateforme complexe. Il doit comprendre en moins de cinq secondes :

- j'appelle ;
- je parle ;
- je rencontre ;
- je rejoins ;
- je peux créer un cercle si besoin.

## Positionnement public

> Le téléchat moderne pour parler, rencontrer et rejoindre des cercles.

Formule courte :

> Simple comme un appel. Protégé comme une plateforme.

## Ce qui reste invisible au premier écran

Ne pas surcharger la home avec :

- marketplace ;
- paiement ;
- analytics ;
- IA ;
- Community OS ;
- infrastructure mondiale ;
- mobile money ;
- monétisation.

Ces couches existent dans l'architecture, les pages dédiées et le back-office, mais la première impression reste humaine : voix, rencontre, appel, WhatsApp, SMS, web.

## Les quatre entrées fondamentales

1. Appel téléphonique
2. WhatsApp
3. SMS
4. Web / mode léger

Aucune application ne doit être obligatoire pour rejoindre la première vague.

## Ambiances prioritaires

- Rencontre
- Amitié
- Diaspora
- Villes
- Business
- Créateurs
- Associations
- Spiritualité

## Créer mon cercle

Un cercle doit générer :

- lien public ;
- QR code ;
- message WhatsApp prêt à partager ;
- lien SMS ;
- bouton appel ;
- page lite ;
- règles de communauté ;
- consentement ;
- future monétisation.

## Architecture durable

Le produit doit être prêt pour :

- PWA ;
- no-JS fallback ;
- vieux Android ;
- petits écrans 320px / 360px / 390px ;
- mode 2G/3G ;
- tel: ;
- sms: ;
- wa.me ;
- IVR / call-in ;
- Supabase ;
- Stripe Europe ;
- mobile money Afrique ;
- traduction vocale future ;
- modération assistée ;
- matching par ville, langue, ambiance et intention.

## Modèle économique sans casser l'âme

La home vend l'usage, pas l'argent.

- Gratuit : rejoindre, parler, écouter.
- Créateur : espace, cercle, salon, paiement futur.
- Association : communauté officielle, annonces, membres.
- Business : présence locale, WhatsApp, appel, coupons.
- Événement : billets, commission, QR.

## Règle de design

Un visiteur doit toujours voir d'abord :

1. Appeler maintenant
2. Rejoindre par WhatsApp
3. Entrer sur le site
4. Créer mon cercle

Tout le reste vient après.
